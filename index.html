// DEVELOPMENT JOURNEY

From idea
to production.

Built solo by Denis Sanchez Leyva — Vertex Coders LLC — Miami, Florida.

PHASE 01 — FOUNDATION
Core Architecture & Gemma 4 Integration
Built the FastAPI backend with Google AI API adapter. Migrated from LM Studio local models to Gemma 4 cloud. Established the multimodal pipeline for text, image, and audio processing.
PHASE 02 — MULTIMODAL
File Processing Engine
Added support for PDFs, DOCX, CSV, JSON, video, and audio files. Integrated OpenCV for video frame extraction, Pandas for data analysis, and Speech Recognition for audio transcription.
PHASE 03 — VISION
Live Camera & Sentinel System
Built real-time vision pipeline with WebSocket streaming. Added MediaPipe fall detection, threat analysis, ASL translation, and the Blind Guide narrator. Sentinel audio/visual alert system deployed.
PHASE 04 — MOBILE
Android & Wear OS Applications
Built VertexCentinel Android app with WebView + native TTS bridge. Developed Vertex Sentinel Wear OS app for Galaxy Watch 5 with real-time vital signs monitoring and Gemma 4 health protocols.
PHASE 05 — PRODUCTION
Global Deployment & Hackathon Submission
Deployed to Netlify (frontend) and HuggingFace Spaces (backend). Published Android and Wear OS apps to Google Play Store. Submitted to Kaggle Gemma 4 Good Hackathon 2026.
